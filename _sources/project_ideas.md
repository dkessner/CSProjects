are thinking of possibilities for immersive, interactive visual and performance
experiences also.

## Visual Novels / Narrative Video Games

We will also learn some basic web development (HTML/CSS/Javascript).  

Last year Senya '23 led the 
[World History Visual Narratives project](http://stem.marlborough.org/blog/history-project/).
This project uses the 
[Monogatari](https://monogatari.io/) 
[visual novel](https://en.wikipedia.org/wiki/Visual_novel) engine.
I used this engine to make
[Alice's Adventures](https://dkessner.github.io/alice/),
